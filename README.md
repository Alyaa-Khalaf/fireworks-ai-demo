# 🤖 Fireworks AI Demo

A ChatGPT-like application with text chat and AI image generation.

## 🌐 Live Demo
👉 [fireworks-ai-demo-production.up.railway.app](https://fireworks-ai-demo-production.up.railway.app)

## ✨ Features
- 💬 **Text Chat** — Powered by Kimi K2 via Fireworks AI
- 🎨 **Image Generation** — AI-generated images from text prompts
- 🔗 **Context Linking** — Image gen understands your chat context
- 📜 **Chat History** — Tracks all conversations and image requests

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Backend:** Node.js, Express
- **AI:** Fireworks AI API (LLM), Pollinations AI (Images)

## 📸 How It Works
1. Type a message in **Text Chat** mode
2. Switch to **Image Gen** mode
3. Type "generate image for it" — the AI understands the context!
